# 🍝 Cosa mangio - PWA Android

Questo pacchetto è pronto per Vercel / Netlify / PWABuilder.

## 1. Incolla il codice app
Apri:

`src/App.jsx`

Cancella il contenuto provvisorio e incolla il codice completo del canvas ChatGPT.

## 2. Installa e prova

```bash
npm install
npm run dev
```

## 3. Build

```bash
npm run build
```

## 4. Pubblica su Vercel
Vai su https://vercel.com e importa questa cartella/progetto.

## 5. Genera APK
Vai su https://www.pwabuilder.com  
Inserisci URL Vercel tipo:

`https://cosa-mangio.vercel.app`

Poi scegli Android package.

## Nota
L'app usa localStorage: funziona offline sullo stesso telefono, ma senza cloud i dati restano su quel dispositivo.
